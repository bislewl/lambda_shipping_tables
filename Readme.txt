This is a Group of 10 Shipping tables that allows you to set different rates for Wholesale & Retail Customers. It also allows you to change the name of the service being offered.

To install
------------

1.) Copy the files to zencart maintiaing the Directory Structure

2.) In the store admin go to modules->Shipping->pick one of the 10 new tables, click install, and start modifying. 

3.) There are two main sets of tables 1 is for retail the other is for wholesale (dual pricing module required for this portion). 

4.) You can rename your shipping method form within the module to make it easier for you (and your customers) to know what each method is. USPS, UPS, Flat Rate, Slow Boat, Custom Critial, White Glove, etc
